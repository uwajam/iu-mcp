# PDF Service

Search API for public academic PDF document chunks.

Current endpoints:

```text
GET  /api/pdf/health
GET  /api/pdf/search?q=卒業要件&limit=10
POST /api/pdf/search
```

The first importer target is:

```text
茨城大学工学部 履修要項 令和8年度入学者用（2026）
https://www.eng.ibaraki.ac.jp/common/education/class/2026-course-registration02.pdf
```
